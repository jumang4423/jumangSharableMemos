# pip3の使い方

python公式のパッケージ管理システム！


インストールしたpythonの中に入っているのでpythonが使える環境であれば、pipは最初から使えます

## pipでできること

モジュール(ライブラリ、便利な関数)やパッケージの
- インストール
- 削除
- バージョン確認
- 一覧表示（インストール済み）
- アップデート
- ダウンロード(インストールはしない)

ができます！

## 使い方
```
pip3 install <パッケージ名>
```
なので、networkxをインストールしたいなら、
```
pip3 install networkx
```
